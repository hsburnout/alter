WELCOME

(This file can be found on the Alter desktop by opening the file Read_Me.txt)

1:

If computer starts acting odd (programs don't open, issues with wifi, or system acting slow), restart it. This will refresh the system and likely solve the problem. Be sure to save your files!

2:

Don't open too many programs at once, it will slow down your system.

3:

The system will occasionally ask for a password. The password will have to be entered to save any unsaved files. If all files are saved, then reboot and you will not be asked for password to login.

4:

'Submission' and 'Alter' shortcuts access the internet so children should be supervised when using them. You need to enter a pin before you can access these features.

5:

Press the windows key to show all open tabs. This key is used to escape from the 'Submission' or 'Alter' feature.

6:

Altercomputers.com is home to a range of tutorials showing you how to use the tools included with the alter.

7:

Go to the 'Shortcut' link at the bottom of altercomputers.com to learn how to get a shortcut to the site on your phone. This will be updated with frequently asked questions so please reach out if you encounter any trouble with your alter and we'll get back to you as soon as we can.
